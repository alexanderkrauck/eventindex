# Privacy Policy — Wasup (Linz Event Index)

**Controller:** Alexander Krauck, alexander.krauck@gmail.com

**What we process:** publicly published event information (titles, dates,
venues, descriptions, prices, links) collected from public websites.
Organizer names appear only as the source published them. We apply
suppression heuristics: events that look private (residential address,
personal organizer) are published without location data.

**What we do NOT process:** no user accounts, no tracking, no cookies, no
profiling of API users. API access logs (IP, timestamp, endpoint) are kept
short-term for abuse prevention only.

**Inferred attributes:** audience estimates (e.g. age range) are clearly
labeled machine estimates with confidence values - never assertions about
individuals.

**Storage & location:** EU-hosted (Germany/Austria). Event claims are kept
as an append-only history for data-quality purposes.

**Your rights (GDPR):** organizers can request correction or removal of
their data (takedown) at the address above; deletion requests are honored
including in the claims history via redaction. Response within 48h.
